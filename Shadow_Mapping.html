Real-Time Shadow Mapping

Shadow mapping is a widely used technique in real-time rendering to create realistic shadows in 3D scenes. I implemented a Real-Time shadow mapping feature for an existing in-house developed Azul game engine as part of my independent study project.

My Work

  • Implemented shaders to support and shadows for spotlight, point-light, and directional light.
  • Used advanced shadow mapping techniques like cube maps and percentage closer filtering (PCF) to improve performance and shadow quality
